WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page Examples of HTML include headings, paragraphs, links, lists, etc.

Check out theHTML tutorial on w3schools.com

Learn CSS

CSS stands for Cascading Style Sheets. CSS is the style of a webpage. Examples of CSS include font styles and colour, background colour, borders, dimensions, layout, etc.

Check out the CSS tutorial on w3schools.com

Learn Javascript

Javascript is the programming language of the web. Javascript helps to make web pages interactive and dynamic. Some common Javascript tasks are:

  • Add new HTML to the page
  • Change the existing content or style
  • React to user events like mouse clicks or key presses

Check out the Javascript tutorial on javascript.info
